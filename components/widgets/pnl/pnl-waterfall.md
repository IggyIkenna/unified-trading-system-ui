# Widget: P&L Waterfall

- **ID:** `pnl-waterfall`
- **Data:** `usePnLData()` — structural rows, factor components, residual, net; `setSelectedFactor` for drill-down.
- **Shared UI:** `CollapsibleSection` for structural block; `PnLValue` for currency.
- **Behavior:** Click a factor bar to select/deselect; residual uses dashed warning styling.
