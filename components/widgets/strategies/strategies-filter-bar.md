# Strategy Filters

Tab: strategies
Widget ID: strategies-filter-bar
Min size: 4 × 1 columns/rows

## What it shows

Search field plus three multi-select dropdowns (asset class, archetype, status) with count badges, removable filter chips, and a clear-all control. Filter options and asset-class colors come from `lib/config/services/strategies.config.ts`.

## Data sources

- `useStrategiesData()` context for filter state and toggles
- Config: `ASSET_CLASS_COLORS`, `ARCHETYPES`, `STATUSES`

## Configuration

- No widget-level configuration options
